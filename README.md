# Mineradio Square

Mineradio Square keeps the Mineradio web UI and swaps the music source layer to the multi-source approach used by MusicSquare.

## Sources

- UI reference: XxHuberrr/Mineradio
- Music source reference: CharlesPikachu/musicsquare
- Integrated providers: Netease, QQ Music, Kuwo, JOOX

## Run

```powershell
npm start
```

The server defaults to `http://localhost:3000`. Set `PORT` to use another port:

```powershell
$env:PORT=3100; npm start
```
